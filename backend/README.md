# HimShakti Backend API Framework

This is the Python/FastAPI backend setup for the HimShakti Generator platform. It manages regional Himalayan product catalog data using structured in-memory storage.

## 🛠️ How to Run Backend Locally (Windows)

Follow these instructions to set up and boot the development server environment locally.

### Prerequisites
* Python 3.10+ installed on your system.

### 1. Set Up the Virtual Environment
Navigate to the backend directory and activate your isolated python sandbox environment:
```powershell
cd backend
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope Process
.\venv\Scripts\Activate.ps1
